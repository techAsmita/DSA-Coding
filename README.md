# 📗 DSA Coding — Data Structures & Algorithms

A comprehensive C++ repository covering **Data Structures and Algorithms** — built assignment by assignment, with additional standalone practice files covering sorting, recursion, linked lists, trees, and heaps.

---

## 📂 Repository Structure

```
DSA-Coding/
├── Assignment 1/      → Arrays (1D & 2D operations)
├── Assignment 2/      → Searching, Sorting & Array Problems
├── Assignment 3/      → Stacks
├── Assignment 4/      → Queues
├── Heap/              → Heap Sort & Priority Queue
├── linked list/       → Singly, Doubly, Circular Linked Lists + 10 Questions
│   ├── SLL/           → Singly Linked List revision
│   ├── DLL/           → Doubly Linked List revision
│   ├── CLL/           → Circular Linked List revision
│   └── questions/     → q1–q10: classic LL problems
└── Standalone files   → Sorting, recursion, trees, sparse matrices
```
